Projects Completed

Malaria Prediction

📌 Malaria Prediction with Machine Learning

In this project, I developed an advanced machine learning model for malaria prediction, leveraging clinical and demographic data. The goal is to identify patients at risk with high accuracy, thus facilitating early and effective management.

🔍 Key approaches used:
✔ Data preprocessing to improve model quality and robustness
✔ Oversampling techniques to manage class imbalance and optimize performance
✔ Interpretability with LIME to explain model decisions and build healthcare professionals' confidence

🚀 Impact: This project contributes to better detection of malaria cases, thus optimizing prevention and treatment efforts in high-incidence areas.

💡 Technologies used: Python, Scikit-learn, Pandas, LIME, Matplotlib, Seaborn.

Breast Cancer Prediction

📌 Early Detection of Breast Cancer with Machine Learning

This project aims to develop a machine learning model capable of detecting breast cancer from medical data. The goal is to help healthcare professionals quickly identify at-risk cases and improve patient care.

🔍 Key approaches used:
✔ Comparison of several algorithms: Logistic Regression, SVM, k-NN, Decision Tree, Naïve Bayes and XGBoost
✔ Analysis of model performance using evaluation metrics such as accuracy, precision, recall and F1-score
✔ Explainability of predictions with LIME to strengthen the confidence of healthcare professionals

🚀 Impact: Thanks to this approach, it is possible to improve early detection of breast cancer and optimize medical decisions, thus reducing the mortality rate associated with this disease. disease.

💡 Technologies used: Python, Scikit-learn, Pandas, Matplotlib, Seaborn, LIME.

📌 Détection Précoce du Cancer du Sein avec le Machine Learning

Ce projet vise à développer un modèle de machine learning capable de détecter le cancer du sein à partir de données médicales. L’objectif est d’aider les professionnels de santé à identifier rapidement les cas à risque et à améliorer la prise en charge des patientes.

🔍 Approches clés utilisées :
✔ Comparaison de plusieurs algorithmes : Régression Logistique, SVM, k-NN, Arbre de Décision, Naïve Bayes et XGBoost
✔ Analyse des performances des modèles à l’aide de métriques d’évaluation telles que l’accuracy, la précision, le recall et le F1-score
✔ Explicabilité des prédictions avec LIME pour renforcer la confiance des professionnels de santé

🚀 Impact : Grâce à cette approche, il est possible d’améliorer le dépistage précoce du cancer du sein et d’optimiser les décisions médicales, réduisant ainsi le taux de mortalité associé à cette maladie.

💡 Technologies utilisées : Python, Scikit-learn, Pandas, Matplotlib, Seaborn, LIME.
